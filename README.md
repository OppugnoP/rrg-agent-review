# Radiology Report Generation Agent: A Comprehensive Review

This repository provides supplementary resources for the survey paper:

> **Radiology Report Generation Agent: A Comprehensive Review**  

## Repository Contents

- [`predict_and_plot.py`](./predict_and_plot.py)  
  Python script to generate trend figures shown in the paper.

- [`paper/`](./paper)  
  - `paper.bib` – BibTeX bibliography file containing 293 studies.
  - `paper_list.csv` – A list of the retrieved papers, specifying the year and title for each entry.
